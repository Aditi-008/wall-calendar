# 🗓️ Interactive Wall Calendar Component

A visually rich and fully interactive **wall calendar component** built using **Next.js + React**, inspired by a physical wall calendar design.

This project focuses on **UI/UX excellence, interactivity, and responsive design**, showcasing frontend engineering skills.

---

## ✨ Features

### 📅 Day Range Selection

* Select start and end dates across the calendar
* Visual highlighting for:

  * Start date
  * End date
  * Dates in between
* Hover preview for smoother selection

---

### 📝 Notes System

* Add **monthly notes**
* Add **per-day notes**
* View all notes in a **pinned notes panel**
* Data persisted using **localStorage**

---

### 🎨 Wall Calendar Aesthetic

* Hero image for each month
* Clean and minimal layout inspired by physical calendars
* Seasonal themes with dynamic colors

---

### 🎴 Interactive UI Enhancements

* Flip animation on hero image
* Smooth transitions between months
* Highlight for current day
* Holiday indicators with legend

---

### 📱 Fully Responsive Design

* **Desktop:** Split layout (calendar + notes panel)
* **Mobile:** Stacked layout optimized for touch interaction

---

## 🛠 Tech Stack

* **Next.js**
* **React**
* **TypeScript**
* **Tailwind CSS**
* **localStorage (for persistence)**

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Aditi-008/wall-calendar.git
cd wall-calendar
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

### 4. Open in browser

```bash
http://localhost:3000
```

## 💡 Design Decisions

* Focused on replicating a **real-world wall calendar experience**
* Used **localStorage** instead of backend for simplicity
* Designed reusable components for scalability
* Emphasized **UX details** like hover states, animations, and visual feedback
